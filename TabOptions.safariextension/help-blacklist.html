Editing the Blacklist

The blacklist is a list of URLs or URL patterns (regular expressions). If the URL of a web page you visit matches one of the entries in the blacklist, Tab Options will ignore keypresses on that page. This is useful for preventing conflicts on web pages that use keyboard shortcuts of their own.

When adding entries to the blacklist, place each entry on its own line. Press Enter to start a new line. Do not type any spaces at the beginning or end of a line.

Each blacklist entry will match any URL that contains that entry. For example, if you add "http://www.google.com/", it will match not only "http://www.google.com/" but also "http://www.google.com/search", "http://www.google.com/images", and every other URL that contains the entry.

Blacklist entries that are regular expressions will match any URL that matches the expression.

An entry does not have to match the beginning of a URL; it can match any part of the URL. For example, you could add "google" to the blacklist, and that would match every Google web page. But be careful, since it would also match a URL like "http://en.wikipedia.org/wiki/google", which may not be what you want.

Hotkeys

Each command in Tab Options can have up to three hotkeys that trigger it. To set a hotkey, click inside one of the three input boxes for the command of interest and press the desired key combination. The hotkey is saved immediately and can be used right away.

A hotkey can be any character key, with or without modifiers such as Command, Option, and Shift. By "character key" we mean any key that produces a visible character if you press it while typing text. (This excludes special keys such as PageDown, Home, NumLock, the arrow keys, etc.)

If you enter a key combination involving the Shift and/or Option keys, what appears in the input box is the character that would appear if you typed that key combination normally. So, for example, if you press Option-Y, you will see the character ¥ in the input box, not the words "Option-Y"; but your hotkey will in fact be Option-Y.
